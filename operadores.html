Operadores Básicos en JavaScript

Los operadores en JavaScript son símbolos que permiten realizar cálculos, comparaciones y operaciones lógicas. Son fundamentales para manipular datos y controlar el flujo de un programa. Hoy veremos algunos de ellos.

Operaciónes Aritméticas

Los operadores aritméticos permiten realizar operaciones matemáticas básicas en variables y expresiones 

let suma = 10 + 8;
let resta = 21 - 5;
let multiplicacion = 3 * 6;
let division = 36 / 6;

Operaciónes Lógicos

Los operadores lógicos permiten combinar condiciones para realizar comparaciones más avanzadas.

  • && (AND): Devuelve true si ambas condiciones son verdaderas.
  • || (OR): Devuelve true si al menos una condición es verdadera.
  • ! (NOT): Invierte el valor de la condición. 
let a = true;
let b = false;
console.log(a && b);   //false

console.log(a || b);   //true

console.log(!a);   //false

Operaciónes de incremento y decremento

Estos operadores permiten aumentar o disminuir el valor de una variable en 1. Usado generalmente en ciclos repetitivos. 

let contador = 5;
contador++;    //6
contador--;   //4

Operadores de Comparación

Los operadores de comparación se utilizan para evaluar condiciones y devuelven true o false. 

console.log(5 == "5");     //Compara ignorando el tipo
console.log(5 == "5");     //Compara ignorando el tipo
console.log(5 === "5");     //Compara el valor y el tipo 
console.log(10 =! "5");     //Diferente en valor
console.log(10 !== "5");     //Diferente en valor y tipo
console.log(5 > 3);     //true(mayor que, tambien existe el mayor o igual que ">=" que incluye el valor comparado )
console.log(5 < 7);     //true(menor que, tambien existe el menor o igual que "<=" que incluye el valor comparado)

Los operadores en JavaScript son herramientas esenciales para realizar cálculos, comparar valores y manejar condiciones. Entender cómo funcionan es clave para escribir código eficiente y lógico.
