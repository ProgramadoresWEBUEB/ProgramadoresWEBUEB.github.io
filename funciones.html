Funciones en JavaScript

Las funciones en JavaScript permiten agrupar código reutilizable y ejecutarlo cuando sea necesario. Son esenciales para organizar y estructurar programas de manera eficiente.

¿Qué es una función?

Una función es un bloque de código que se define una vez y se puede ejecutar múltiples veces. Se puede pasar información a la función a través de parámetros y, opcionalmente, devolver un resultado.

Declaración de una función

Las funciones en JavaScript se pueden definir de varias maneras. La más común es la función declarada: 

saludar = function () {
console.log("Hola Mundo :D");
}
Para ejecutar la función, debemos llamarla: 
saludar();  //Output: Hola Mundo :D

Funciones con parámetros

Las funciones pueden recibir datos a través de parámetros. 

suma = function (numero1,numero2) {
    return numero1 + numero2;}
}
let resultado = suma(5,3);
console.log(resultado);     //Output: 8

Funciones con valores por defecto

Podemos asignar valores por defecto a los parámetros en caso de que no se pasen argumentos. 

resta = function (numero1,numero2) {
    return numero1 - numero2;}
}
let resultado = resta(5,3);
console.log(resultado);     //Output: 2

Las funciones en JavaScript son una herramienta clave para la reutilización de código y la estructuración de programas. Conocer sus diferentes formas y usos permite escribir código más eficiente y modular.
