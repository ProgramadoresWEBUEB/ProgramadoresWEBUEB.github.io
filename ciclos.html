Ciclos Repetitivos o Bucles en JavaScript

Los bucles en JavaScript permiten ejecutar repetidamente un bloque de código mientras se cumpla una condición. Son fundamentales para la automatización de tareas repetitivas, como recorrer listas o ejecutar acciones múltiples veces.

Bucle for

El bucle for es ideal cuando sabemos cuántas veces queremos repetir un bloque de código. Su estructura básica es: 

for (inicializacion;condicion;incremento) {
    //código a ejecutar
}

Bucle while

El bucle while se ejecuta mientras una condición sea verdadera. Se usa cuando no sabemos exactamente cuántas iteraciones serán necesarias. 

let contador = 0;
while (contador < 5) {
    //código a ejecutar
   contador++;
}

Bucle do...while

El bucle do...while es similar al while, pero garantiza que el código se ejecuta al menos una vez antes de evaluar la condición. 

let contador = 0;
do{
    //código a ejecutar
   contador++;
}while (contador < 5)

Los bucles en JavaScript permiten ejecutar código repetitivo de manera eficiente. Elegir el tipo adecuado (for, while, do...while,) es clave para escribir código claro y funcional.
