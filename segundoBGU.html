Hechos y conceptos

  • Conceptos básicos de web

    HTML, páginas web, páginas estáticas, páginas dinámicas, protocolos IP, navegadores (browsers), hojas de estilo, tablas, plantillas (template), marcos (frames), hipervínculos locales y externos, software de edición de sitios web, publicador de sitio web, servidor, dominio, alojamiento y publicación, solución web, plataforma web.

    • HTML (HyperText Markup Language): Lenguaje de marcado estándar utilizado para crear la estructura de una página web.

    • Páginas web: Documentos digitales accesibles a través de internet mediante un navegador.

    • Páginas estáticas: Sitios web que muestran el mismo contenido a todos los usuarios sin interacción con bases de datos.

    • Páginas dinámicas: Sitios web que generan contenido en tiempo real mediante bases de datos y lenguajes de programación del lado del servidor (PHP, JavaScript, Python, etc.).

    • Protocolos IP (Internet Protocol): Conjunto de reglas que permiten la comunicación entre dispositivos en una red.

    • Navegadores (browsers): Programas que interpretan y muestran el contenido de las páginas web (Google Chrome, Mozilla Firefox, Microsoft Edge, Safari, etc.).

    • Hojas de estilo (CSS): Lenguaje utilizado para definir la presentación y el diseño de las páginas web.

    • Tablas: Elementos HTML utilizados para organizar datos en filas y columnas dentro de una página web.

    • Plantillas (templates): Diseños predefinidos que facilitan la creación de páginas web con una estructura y estilo coherentes.

    • Marcos (frames): Elementos HTML antiguos que permitían dividir una página en varias secciones independientes (ahora en desuso por problemas de accesibilidad y SEO).

    • Hipervínculos locales y externos: Enlaces que conectan una página web con otras dentro del mismo sitio o con páginas externas en internet.

    • Software de edición de sitios web: Herramientas como Adobe Dreamweaver, Visual Studio Code y WordPress que permiten crear y modificar páginas web.

    • Publicador de sitio web: Aplicaciones y servicios que permiten subir y gestionar un sitio web en línea, como FTP y gestores de contenido (CMS).

    • Servidor: Computadora o sistema que almacena y distribuye páginas web a los usuarios a través de internet.

    • Dominio: Nombre único que identifica un sitio web en internet (ejemplo: www.misitio.com).

    • Alojamiento y publicación: Espacio en servidores donde se almacenan los archivos de un sitio web y su proceso de publicación en internet.

    • Solución web: Aplicación o conjunto de servicios diseñados para resolver necesidades específicas en línea (tiendas virtuales, blogs, plataformas educativas, etc.).

    • Plataforma web: Entorno digital que permite la creación, administración y distribución de contenido en línea (CMS, LMS, foros, redes sociales, etc.).

  • Conceptos básicos de herramientas multimedia

    Herramientas de dibujo y pintura, mapas de bits, dibujos vectoriales, animaciones en 2D y 3D, edición de fotografía, edición de video no lineal, software de edición gráfica y edición de sonido.

    • Herramientas de dibujo y pintura: Software utilizado para la creación de ilustraciones digitales (Adobe Illustrator, CorelDRAW, Krita, etc.).

    • Mapas de bits: Imágenes compuestas por píxeles (JPEG, PNG, BMP, GIF).

    • Dibujos vectoriales: Gráficos creados a partir de líneas y curvas matemáticas, escalables sin perder calidad (SVG, AI, EPS).

    • Animaciones en 2D y 3D: Creación de contenido animado con software como Adobe Animate, Blender, Toon Boom, Maya, etc.

    • Edición de fotografía: Procesamiento y retoque de imágenes digitales (Adobe Photoshop, GIMP, Lightroom).

    • Edición de video no lineal: Manipulación de secuencias de video sin alterar el archivo original (Adobe Premiere Pro, DaVinci Resolve, Final Cut Pro).

    • Software de edición gráfica: Herramientas utilizadas para la creación y manipulación de imágenes digitales.

    • Edición de sonido: Programas para la grabación y modificación de audio (Audacity, Adobe Audition, FL Studio).

  • Publicación y promoción de soluciones web

    • Redes sociales, plataformas web, buscadores, programas de posicionamiento y enlaces desde sitios externos, soporte de las soluciones web. Costos de publicación.

    • Redes sociales: Uso de plataformas como Facebook, Instagram, Twitter, TikTok para difusión.

    • Plataformas web: Publicación en blogs, foros y comunidades en línea.

    • Buscadores: Optimización para motores de búsqueda (SEO).

    • Programas de posicionamiento y enlaces desde sitios externos: Estrategias de link building y publicidad digital.

    • Soporte de las soluciones web: Mantenimiento y actualización continua del sitio.

    • Costos de publicación: Evaluación de inversión en hosting, dominios y publicidad.

    Procedimientos

    • Describir los diferentes elementos y procedimientos para estructurar una página web de acuerdo a determinados estándares de calidad.

    • Utilizar diferentes herramientas multimedia para la generación de procesos de animación y edición, en concordancia con los requerimientos que se establezcan.

    • Diseñar y elaborar prototipos de soluciones web con interfaces fáciles de comprender y de usar, cumpliendo los parámetros establecidos.

    • Elaborar páginas web integrando elementos multimedia y aplicando los diversos estándares de calidad requeridos en el prototipo aprobado.

    • Implementar los cambios requeridos a nivel de código fuente, interfaz, esquemas de actualización del servidor, alojamiento y publicación, reconociendo sus fortalezas y debilidades.

    Actitudes, valores y normas

    • Trabajar de forma autónoma en la planificación y desarrollo de soluciónes web, asumiendo las acciones encomendadas con responsabilidad, eficiencia, ética y disciplina.

    • Mostrar interés e iniciativa en la proposición de ideas que permitan generar soluciones web de acuerdo a los requerimientos del usuario u organización.

    • Tener una actitud propositiva para aportar con ideas creativas e innovadoras, respetando los derechos de autoría y aplicando normas, estándares y especificaciones técnicas.

    • Mostrar interés por mantenerse actualizado en temas relacionados con el diseño y desarrollo de soluciones web.

    • Ser flexible y adaptarse a los cambios y nuevas situaciones en el entorno de trabajo.
