Variables y tipos de datos en JavaScript

JavaScript es un lenguaje de programación dinámico que permite definir variables para almacenar datos. Las variables pueden contener diferentes tipos de información y su correcta gestión es clave para el desarrollo web. En este artículo, exploraremos cómo declarar variables y los tipos de datos en JavaScript.

Declaración de Variables

  1. var (Variable Global o de Función)

    Características:

    • Tiene ámbito de función, es decir, si se declara dentro de una función, solo existe dentro de ella.
    • Puede ser redeclarada en el mismo ámbito, lo que puede causar errores inesperados. 
    var nombre = "Juan";
    console.log(nombre);

  2. let (Variable de Bloque)

    Características:

    • Su ámbito es bloque, es decir, solo existe dentro del {} donde se define.
    • No permite ser redeclarada en el mismo bloque, evitando errores. 
    let edad = 25;
    console.log(edad);

  3. const (Constantes)

    Características:

    • Su valor no puede ser modificado después de la asignación.
    • También tiene ámbito de bloque. 
    cost PI = 3.1416;
    console.log(PI);

Tipos de datos

  1. Tipo String (Cadena de texto)

    Los String representan texto y se definen con comillas simples ('), dobles (") o backticks (`). 
    let mensaje = "Hola, mundo!"; 
    console.log(mensaje);

  2. Tipo Number (Número)

    Los Number incluyen enteros y decimales(int,float). 
    let entero = 42; 
    let decimal = 3.14; 
    console.log(entero,decimal);

  3. Tipo Boolean (Boleano)

    Los Boolean representan valores true o false. 
    let esMayor = true; 
    console.log(esMayor);

  4. Tipo Null y Undefined

    null: Representa la ausencia de valor. undefined: Se asigna automáticamente a una variable no inicializada. 
    let sinValor = null; 
    let noDefinido; 
    console.log(sinValor,noDefinido);
