Fundamentos HTML

Estructura básica de HTML

HTML (HyperText Markup Language) es el lenguaje fundamental para la creación de páginas web. Comprender su estructura básica es esencial para construir documentos bien organizados y funcionales. A continuación, exploraremos los principales elementos que componen un documento HTML.

1. Declaración del Tipo de Documento

Cada archivo HTML debe comenzar con una declaración de tipo de documento (DOCTYPE), que le indica al navegador que está interpretando un documento HTML: 

<!DOCTYPE html>

Esto asegura que el navegador renderice el documento de acuerdo con los estándares modernos de HTML5.

2. La Estructura General de un Documento HTML

Un archivo HTML está organizado en varias secciones principales: 

<!DOCTYPE html> 
<html lang="es"> 
<head > 
  <meta charset="UTF-8"> 
  <meta http-equiv="X-UA-Compatible"content="IE=edge"> 
  <meta name="viewport"content="width=device-width, initial-scale=1.0"> 
  <title> Ejemplo </title> 
</head> 
<body> 
  <h1> Un clasico ejemplo </h1> 
  <p> Hola Mundo </p> 
</body> 
</html>

A continuación hablaremos acerca de cada parte de esta estructura

3. Etiqueta <html>

Toda la página web se encuentra dentro de la etiqueta . El atributo lang especifica el idioma principal del documento, en este caso, español (es).

3. El header o head<head>

La sección contiene metadatos y enlaces a recursos externos, como hojas de estilo o scripts. Algunos elementos importantes dentro de son:

  • <meta charset="UTF-8"> : Define la codificación de caracteres como UTF-8, lo que permite el uso de caracteres especiales.
  • <meta name="viewport"content="width=device-width, initial-scale=1.0">: Garantiza una correcta visualización en dispositivos móviles.
  • <title>:Define el título de la página, que aparece en la pestaña del navegador.

4. Cuerpo o body <body>

La sección <body> contiene el contenido visible de la página, como texto, imágenes, vídeos y formularios. Dentro del <body> podemos incluir:

  • Encabezados:

    <h1> a <h6> para estructurar jerárquicamente el contenido.

  • Párrafos:

    <p> para definir bloques de texto.

  • Listas:

    <ul> (lista desordenada) y <ol> (lista ordenada) con elementos <li>(viñetas).

  • Imágenes:

    <img src="URL" width="Descripción"> para insertar gráficos.

Conocer la estructura básica de un documento HTML es el primer paso para el desarrollo web. A partir de esta base, se pueden agregar estilos con CSS y funcionalidades interactivas con JavaScript para mejorar la experiencia del usuario.
