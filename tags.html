Fundamentos HTML

Etiquetas esenciales en HTML

HTML (HyperText Markup Language) es el lenguaje de marcado que define la estructura de una página web. Para construir documentos web funcionales, es fundamental conocer las etiquetas esenciales. A continuación, exploramos las más importantes y su uso.

1. Etiquetas de texto

El texto es un elemento clave en cualquier página web. HTML proporciona varias etiquetas para estructurar y dar formato al contenido:

  • Encabezados:

    Definen la jerarquía del contenido, desde el más importante (h1) hasta el menos relevante (h6).

    <h1>Título</h1>
    <h2>Subtítulo</h2>
    <h3>Sección</h3>

    A continuación los resultados:

    Título

    Subtítulo

    Sección

  • Párrafos:

    Se crean con la etiqueta <p>.

    <p>Para entender la historia de Five Nights at Freddy's hay que olvidarse que estos son juegos y quiero que tomen realmente a esta saga como lo que es. ¿Terror? Sí, pero sobre todo, ciencia ficción.</p>

    A continuación los resultados del ejemplo:

    Para entender la historia de Five Nights at Freddy's hay que olvidarse que estos son juegos y quiero que tomen realmente a esta saga como lo que es. ¿Terror? Sí, pero sobre todo, ciencia ficción.

3. Enlaces e hipervínculos

Para enlazar otras páginas o recursos, usamos<a> (usare un video para el ejemplo): 

<a href="https://youtu.be/Zz_dnrxXu3I?si=mFBUCpuc4yMiYipc">I´m a vocaloid chat :O</a>

Podemos hacer que el enlace se abra en una nueva pestaña usando target="_blank": 

<a href="https://youtu.be/Zz_dnrxXu3I?si=mFBUCpuc4yMiYipc"target="_blank">I´m a vocaloid chat :O but in other tab</a>

A continuación el resultado de los ejemplos (perdón por no ser serio jajaja):

I´m a vocaloid chat :O i´m Case - UTAU >:D
I´m a vocaloid chat :O i´m Case - UTAU >:D (in other tab)

4. Listas

Las listas permiten organizar información de manera estructurada. Existen 2 tipos:

  • Listas desordenadas:

    <ul> 
       <li>Elemento 1</li>
       <li>Elemento 2</li>
    </ul>

  • Listas ordenadas:

    <ol> 
       <li>Paso 1</li>
       <li>Paso 2</li>
    </ol>
El ejemplo son estas listas en si mismas.

Estas etiquetas son la base para construir cualquier sitio web en HTML. Con ellas, podemos estructurar y presentar información de manera organizada y accesible.
