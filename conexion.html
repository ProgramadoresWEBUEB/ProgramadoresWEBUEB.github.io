Fundamentos HTML

Conexiones HTML a CSS y JS

HTML por sí solo nos permite estructurar una página web, pero para hacerla visualmente atractiva y funcional, necesitamos integrar CSS para los estilos y JavaScript para la interactividad.

1. ¿Qué rol desempeñan?

  • HTML (HyperText Markup Language): Define la estructura del contenido.
  • CSS (Cascading Style Sheets): Agrega estilos y diseño.
  • JavaScript: Añade interactividad y lógica.

2. ¿Cómo integrar CSS a tu página HTML?

Hay bastantes formas de hacerlo pero nosotros te hablaremos de 2:

  • Dentro de una línea

    Se usa el atributo style directamente en una etiqueta HTML. 
    <p style="color: blue; font-size: 18px;">Este texto es azul :D.</p>

  • Externo

    Se recomienda enlazar una hoja de estilos separada con <link>. 
    <head>
    <link rel="stylesheet"href="hoja.css">
    </head>

3. ¿Cómo integrar JS a tu página HTML?

Como en CSS, hay varias formas de hacerlo pero nosotros te mostraremos las que nos parecen más sencillas:

  • Dentro de una línea

    Se usa el atributo onclick para manejar eventos directamente en el HTML. 
    <button onclick="alert("Hola mundo!")">Click!</button>

  • Externo

    Se recomienda enlazar una hoja de estilos separada con <link>. 
    <head>
    <script src="funciones.js"></script>
    </head>

Usar HTML, CSS y JavaScript en conjunto permite crear sitios web completos. HTML proporciona la estructura, CSS mejora la apariencia y JavaScript añade interactividad.
